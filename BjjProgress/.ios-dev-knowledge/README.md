# iOS Development Knowledge Base

This directory contains curated Apple documentation and best practices for iOS app development.

## Structure

```
.ios-dev-knowledge/
├── design-principles/       # Apple design guidelines and HIG
├── app-store-guidelines/    # App Store review guidelines
├── common-rejections/       # Common rejection reasons and fixes
├── technical-setup/         # Technical implementation guides
└── checklists/             # Pre-submission and review checklists
```

## How to Use

1. **Before Starting a New App:** Review design principles and HIG
2. **During Development:** Reference technical setup guides
3. **Before Submission:** Use pre-submission checklist
4. **After Rejection:** Check common rejections for solutions
